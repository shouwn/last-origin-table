# last-origin-table

라스트 오리진 효율표

## 사용방법

1. 자신이 탐색 보낼 수 있거나 보내고 싶은 지역을 선택
2. 탐색을 연속해서 보낼 것인지 1번만 보낼 것인지 선택 (탐색 반복 버튼)
3. 정렬 기준을 선택 - 어떤 자원 순으로 정렬을 할 것인지
4. 아래 표를 보면서 자신에게 맞는 탐색을 돌림
4.1 상세를 보고 싶으면 각 row를 클릭해서 상세를 볼 수 있음
